## 🧠 核心功能

- 📈 即時能耗圖表：幫你揪出家中最兇的電力吸血鬼  
- 🚨 高耗能預警：冷氣忘了關？我們幫你盯  
- 🧠 簡易診斷模型：比你爸更懂省電  
- 📄 一鍵報告：KPI 交差不再靠嘴說

## ⚙️ 快速開始

```bash
git clone https://fuyewd.github.io/KOKO/
cd energy-analyzer
npx serve .        # or python3 -m http.server
````
## 🧩 技術棧（簡明扼要）

* `Chart.js`：互動圖表
* `LocalStorage API`：儲存即時數據
* `自製能耗算法`：給電表裝上腦子
* `純前端`：輕巧、即插即用
